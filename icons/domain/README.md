# 🌐 Domain

The `domain` folder contains **domain-related icons and assets** used throughout the application.

---

- All icons are in **SVG format**.
- These icons are generally used in:
  - External Hosts
  - Inspector
  - Inventory
  - Timeline Secondary
- For new domain types:
  - Add the SVG directly here until the structure requires sub-folders.
  - Avoid duplicates; reuse existing icons when applicable.
