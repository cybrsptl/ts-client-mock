# 🎨 UI Core

This folder contains **core UI icon sets** in SVG format.  
These assets are used across the application for UI elements such as navigation, status indicators, priorities, and metrics.

---

## Structure

The folder is organized into sub-folders based on icon usage:

- **`collection/`**  
  General-purpose icons currently used in development.

- **`file_types/`**  
  Icons that represent different file types (e.g., PDF, CSV, DOC).

- **`key_events_and_metrics/`**  
  Icons used for significant events, triggers, and telemetry metrics.

- **`navigation/`**  
  Icons for navigation components (menus, sidebars, breadcrumbs, etc.).

- **`priority/`**  
  Icons indicating levels of task or incident priority.

- **`severity/`**  
  Icons used to show levels of severity (e.g., warning, critical, info).

- **`status/`**  
  Icons to visually indicate state or progress (e.g., active, inactive, success, error).

- **`teleseer_ui/`**  
  Icons specific to the **Teleseer** interface.

- **`unused/`**  
  Deprecated or unreferenced icons that are currently not in use but retained for reference or future migration.

---
- All icons are **SVG format** for scalability and crisp rendering.
- Use icons from the appropriate category to maintain **consistency and semantic meaning**.
- Before adding new icons:
  - Follow the **existing design language** (size, stroke, color).
  - Follow the **naming convention** snake case.
  - Place them in the most relevant sub-folder.
- Consider cleaning or reviewing the `unused/` folder periodically to avoid clutter.
