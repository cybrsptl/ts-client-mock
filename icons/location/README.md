# 📍 Location

The `location` folder contains **SVG assets used for representing geographical locations and places**.  
These icons are commonly used in maps, dashboards, and location-based visualizations.

---

## Structure

The folder is organized into two main sub-folders:

- **`country_flags/`**  
  Contains flag icons for supported countries.

- **`place/`**  
  Contains icons for places, landmarks, or location-based markers (e.g., buildings, cities, pins).

---

- All icons are in **SVG format**.
- Typically used in:
  - Geolocation-based features
  - Country/region selectors
  - Dashboards and analytics maps
- When adding new assets:
  - Follow existing **file naming conventions** (e.g., `us.svg`, `ph.svg` for flags, `city-pin.svg` for place markers).
  - Keep icons **optimized and lightweight** for performance.
  - Ensure visual **consistency** (size, stroke, and style) with existing assets.
- Avoid duplicating assets — reuse existing flags or place icons whenever possible.

---

## Maintenance
- Periodically review the folder to remove unused or outdated icons.
- Keep the **flag designs updated** to match current official standards if geopolitical changes occur.
