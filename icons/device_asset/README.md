# 📦 Device Asset

This folder contains **isometric SVG icons** that visually represent various types of devices.  
These icons are used throughout the project for consistent and recognizable device representations.

---

## Structure

The folder is organized into sub-folders based on device categories:

- **`container/`**  
  Icons representing containerized services or container-based infrastructure.

- **`device/`**  
  General device icons such as hardware endpoints or network devices.

- **`operating_system/`**  
  Icons representing operating systems (e.g., Linux, Windows, macOS).

- **`server/`**  
  Icons representing physical and virtual server hardware.

- **`virtual_machine/`**  
  Icons for virtualized environments and hypervisors.
